Quick Start

Up and running in under a minute

Install from npm or load from CDN — choose the method that fits your workflow.

Install via npm
npm install @scaleflex/crop
<script type="module">
  import '@scaleflex/crop/define';
</script>

<sfx-crop
  src="https://example.com/photo.jpg"
  crop-shape="free"
></sfx-crop>
Use via CDN
<script type="module"
        src="https://esm.sh/@scaleflex/crop/define"></script>

<sfx-crop src="/photo.jpg" crop-shape="free"></sfx-crop>
